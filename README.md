# Customer-Spending-Behavior-Analysis-Power_BI

## Project Overview

This project analyzes customer demographic and behavioral data from an imaginative retail shop to identify spending patterns and understand the characteristics of ideal customers.

The shop collects customer information through **membership cards**, which include demographic details and purchasing behavior. Using this data, a **Power BI dashboard** was created to help the business better understand its customers and support data-driven marketing strategies.

The dataset contains **2000 customer records and 8 attributes**, allowing analysis of customer behavior based on demographic and professional factors.

---

## Business Problem

Retail businesses often struggle to identify **which customers contribute the most to revenue** and how different customer segments behave.

The shop owner wants to better understand customer behavior in order to:

* Identify **high-spending customer segments**
* Understand how **income, age, and profession affect spending**
* Design **targeted marketing campaigns**
* Improve **customer retention strategies**

To address these challenges, a **Power BI dashboard** was built to analyze customer spending patterns and demographic insights.

---

## Dataset Description

The dataset represents customer information collected through **shop membership cards**. It contains **2000 records and 8 columns**.

### Dataset Features

| Column                 | Description                                                               |
| ---------------------- | ------------------------------------------------------------------------- |
| Customer ID            | Unique identifier for each customer                                       |
| Gender                 | Gender of the customer                                                    |
| Age                    | Age of the customer                                                       |
| Annual Income          | Annual income of the customer                                             |
| Spending Score (1–100) | Score assigned by the shop based on customer behavior and spending nature |
| Profession             | Occupation of the customer                                                |
| Work Experience        | Total years of professional experience                                    |
| Family Size            | Number of family members in the customer's household                      |

## Tools Used

* **Power BI** – Data visualization and dashboard creation
* **Power Query** – Data cleaning and transformation
* **CSV Dataset** – Data source

---

## Dashboard Features

The interactive dashboard provides insights into customer demographics and spending patterns, including:

* Total Customers
* Average Annual Income
* Average Family Size
* Average Spending Score
* Average Work Experience
* Spending Score by Gender
* Spending Score by Family Size
* Spending Score by Profession
* Spending Score by Experience Level
* Customer Distribution by Age

Users can filter the dashboard by:

* Gender
* Profession
* Experience Level
* Family Size

---

## Key Insights

* Male and female customers have **almost identical average spending scores**.
* Customers working in **Entertainment and Artist professions** show higher spending scores.
* Spending behavior varies across **different age groups**.
* **Entry-level and junior professionals** show slightly higher spending patterns.
* Family size has a moderate influence on customer spending behavior.

---

## Business Recommendations

Based on the analysis:

* Focus marketing campaigns on **high-spending professions**.
* Create **age-based targeted promotions**.
* Use customer segmentation to improve **marketing strategies**.
* Offer loyalty programs for customers with **higher spending scores**.
